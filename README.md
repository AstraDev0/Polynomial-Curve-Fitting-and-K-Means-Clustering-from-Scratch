## 🧪 Problem 1: Polynomial Curve Fitting

- Implements curve fitting for:
  - (a) Polynomial of degree **2**
  - (b) Best-fit polynomial based on visual/quantitative criteria
  - (c) Comparison of fit errors for **m = 3** and **m = 8**
- Input datasets: `x.txt`, `y1.txt`
- Output: regression plots (inferred to be shown in the PDF)
- 🔧 Implemented without built-in fitting functions like `numpy.polyfit`

## 🧪 Problem 2: Clustering

- Implements from scratch:
  - (a) K-Means (K=5) with specified initialization
  - (b) K-Means with alternative initialization
  - (c) Non-Uniform Binary Split
- Input dataset: `Dataset_2.txt`
- Results visualized in clusters (plots assumed shown in `.pdf`)
- Fixed color map for clusters:
  - 1: Blue
  - 2: Black
  - 3: Red
  - 4: Green
  - 5: Magenta

## 📌 Notes
